# 散线转文字 - 使用说明

## 功能介绍

一键将CAD中的散线文字转换为可编辑的文字对象，基于 PaddleOCR 深度学习引擎，识别准确率高达 98%。

## 安装步骤

1. 下载安装包，直接安装即可。最好安装到C盘以外的盘
2. 启动 CAD，输入命令 `SXZWZ_LXB` 即可使用

## 使用方法

1. 在 CAD 中打开包含散线文字的图纸
2. 输入命令 `SXZWZ_LXB`
3. 框选需要识别的散线文字区域
4. 等待识别完成，文字将自动替换为可编辑文字对象

## 注意事项

- 识别精度取决于散线文字的清晰度
- 建议在识别前对图纸进行适当缩放
- 支持中英文混合识别

## 常见问题

**Q: 识别结果不准确怎么办？**
A: 尝试提高散线文字的分辨率，或使用 CAD 的缩放功能放大后再识别。

**Q: 支持哪些 CAD 版本？**
A: 支持 AutoCAD 2018 及以上版本。
