# 1、安装说明

## 1.1 版本支持

| 项目 | 说明 |
|------|------|
| 支持平台 | AutoCAD 2014 ~ 2024 |
| 运行环境 | .NET Framework（随 CAD 自带） |
| 安装方式 | 一键安装程序（Setup.exe） |

## 1.2 安装工具箱

安装以及相关教学视频：[【【自研】AutoCAD插件：打造你自己的CAD神器合集（不加班的小刘_工具箱安装教程）]( https://b23.tv/Idgt0sk)


**安装前请务必关闭所有 AutoCAD 进程。**

1. 双击安装程序，点击「下一步」进入安装向导。

![[file-20260403110949824.png|300]]

2. 选择安装路径，建议将默认 `C:\` 盘路径修改为 `D:\` 盘。

![[file-20260403111032246.png|300]]

3. 勾选同意协议，点击「安装」，等待约 2 分钟完成安装。

![[file-20260403111142988.png|300]]


![[file-20260403111305223.png|300]]


> **注意：** 安装路径可以自由选择，但路径末尾的文件夹名称必须为 `不加班的小刘_工具箱`。

## 1.3 安装CAD字体（强烈建议安装，否则部分功能因为字体导致使用不了）

安装目录下包含 `Fonts` 文件夹，内含 CAD 所需字体文件。请将 `Fonts` 文件夹中的所有字体复制到 AutoCAD 安装目录的 `Fonts` 文件夹下。

具体操作：`安装目录\Fonts\*.*` → `AutoCAD安装目录\Fonts\`

## 1.4 安装目录结构

安装完成后，目录结构如下：

```
不加班的小刘_工具箱/
├── 不加班的小刘_工具箱配置.xlsx   # 外部工具配置文件（Excel格式）
├── Image/                         # 外部图标文件夹
├── lisp/                          # LISP 脚本文件夹
├── dll/                           # .NET DLL 插件文件夹
├── arx/                           # ARX 原生扩展文件夹
└── Fonts/                         # CAD 字体文件夹
```

# 2、使用说明

## 2.1 启动与授权

### 2.1.1 启动工具箱

打开 AutoCAD，在命令行输入以下命令：

```
BJB
```

CAD 启动时会自动加载工具箱插件，命令行提示 `不加班的小刘_工具箱已加载` 即表示加载成功。

### 2.1.2 授权说明

首次运行 `BJB 命令时，系统会弹出授权窗口：


![[file-20260403112340782.png|400]]

1. 授权窗口会显示本机的 **机器码**。
2. 将机器码发送给开发者，获取对应的 **授权码**。
3. 将授权码粘贴到授权窗口中，点击确认即可完成授权。

> **授权规则：**
> - 采用 **一机一码** 授权机制，一个授权码仅对应一台电脑。
> - 如需多台电脑使用（最多3台），请联系开发者备注。
> - 工具箱底部会显示授权剩余时间，授权到期前 7 天会变为橙色提醒（到期联系开发授权）。

**开发者联系方式：**

####  2.3.1.2 图框编辑

**功能：** 批量编辑图框图块的属性值，包括项目名称、日期、设计阶段、图号和图名。

建议使用开发者提供的图框进行修改。

3. 在弹出的编辑界面中：
   - **左侧** — 修改项目名称、日期、设计阶段（等等这里的属性是根据，图块中有的属性去读取的）。
   - **右侧** — 设置图号前缀、起始编号，点击「生成图号」自动编号。
   - **下方** — 逐个编辑图名。
4. 确认后批量应用修改。

图名属性标签：指的意思就是ASDHK-DSA-020-001-SAD这个位置所在的属性名称。比如我这个ASDHK-DSA-020-001-SAD这个属性字的属性名称为图号，那么下方这个属性标签中应该输入图名。

## 2.4 外部工具扩展

工具箱支持通过 **Excel 配置文件** 动态加载外部工具，无需修改代码即可扩展功能。

### 2.4.1 配置文件说明

配置文件位于安装目录下：`不加班的小刘_工具箱配置.xlsx`

![[image-2.png|500]]

Excel 文件必须包含以下 6 列（首行为表头）：

| 列名   | 说明                  | 示例             |
| ---- | ------------------- | -------------- |
| 分类   | 工具所属分类名称            | 常用工具           |
| 工具名称 | 显示在按钮上的名称           | 图框编辑           |
| 命令名称 | AutoCAD 命令          | TKBJ           |
| 文件路径 | 相对于子文件夹的路径          | 0-图框编辑/xxx.lsp |
| 文件类型 | 文件格式类型（见下表）         | LISP           |
| 图标   | 图标文件名（放于 Image 文件夹） | 图框编辑.png       |

### 2.4.2 支持的文件类型

| 类型标识 | 说明 | 文件存放路径 |
|----------|------|-------------|
| `CS` | C# 内置命令 | 编译在 DLL 中 |
| `BUILTIN` | 纯内置命令（无外部文件） | — |
| `DLL` | .NET DLL 动态加载插件 | `dll/` 文件夹 |
| `LISP` | AutoLISP 脚本 | `lisp/` 文件夹 |
| `ARX` | AutoCAD ARX 原生扩展 | `arx/` 文件夹 |

### 2.4.3 热更新机制

工具箱内置了 **配置文件热更新** 功能：

- 系统会自动监听 `不加班的小刘_工具箱配置.xlsx` 文件的变化。
- 当检测到配置文件被修改后，会弹出提示框询问是否立即刷新工具列表。
- 选择「是」即可重新加载所有工具，**无需重启 CAD**。

也可在命令行手动执行刷新命令：

```
BJB_RELOAD
```

## 2.5 主题设置

点击工具箱底部的「设置」按钮，即可打开主题设置窗口，自定义工具箱的外观样式。

### 2.6.1 预置主题

工具箱内置了 **9 套精心设计的主题**，涵盖不同风格场景：

| 主题名称 | 风格标签 | 适用场景 |
|----------|----------|----------|
| 默认白色 | 简洁清新 | 日常通用，搭配浅色 CAD 界面 |
| 深色模式 | 护眼舒适 | 长时间绘图，降低视觉疲劳 |
| 护眼绿 | 自然温和 | 模拟纸质色调，缓解用眼压力 |
| 科技蓝 | 专业稳重 | 办公环境，搭配蓝色系 CAD 界面 |
| 霓虹紫 | 赛博朋克 | 深色背景 + 紫色发光边框 + 渐变效果 |
| 烈焰红 | 热情活力 | 深色背景 + 红色按钮发光 + 辉光边框 |
| 暮光橙 | 温暖时尚 | 深色背景 + 橙色头部渐变效果 |
| 薄荷绿 | 清新现代 | 浅绿色调 + 大圆角按钮 + 柔和阴影 |
| 星空黑 | 高端大气 | 深色背景 + 蓝色发光 + 渐变背景 + 辉光边框 |

**操作方式：** 在设置窗口中点击目标主题卡片，底部会显示当前选中的主题名称，点击「应用主题」即可立即生效。

### 2.6.2 自定义主题编辑器

如预置主题无法满足需求，可点击设置窗口中的「自定义主题编辑器」按钮，打开专业级主题编辑器。

![[image-4.png|500]]

**编辑器布局：**

| 区域 | 功能 |
|------|------|
| **顶部** | 主题名称输入框 + 「基于预设」下拉选择（可基于已有主题修改） |
| **左侧编辑区** | 按「面板 / 头部 / 分类卡片 / 工具按钮 / 底部 / 辅助」分类的颜色属性列表 |
| **右侧预览区** | 工具箱实时预览面板，修改颜色后即时看到效果 |
| **底部操作栏** | 导出主题 / 导入主题 / 取消 / 保存并应用 |

**可自定义的颜色属性（共 17 项）：**

- **面板：** 面板背景色、面板边框色
- **头部：** 头部背景色、头部边框色、头部文字色
- **分类卡片：** 卡片背景色、卡片边框色、分类文字色、箭头颜色
- **工具按钮：** 按钮背景色、按钮悬停色、按钮文字色
- **底部：** 底部背景色、底部边框色、底部文字色
- **辅助：** 滚动条颜色、分隔线颜色

**高级视觉效果参数：**

| 参数 | 说明 |
|------|------|
| 渐变背景 | 面板整体使用顶部→底部的渐变色 |
| 头部渐变 | Logo 区域启用渐变效果 |
| 按钮发光 | 工具按钮外圈发光颜色 |
| 边框发光 | 面板边框辉光效果及强度 |
| 阴影不透明度 | 卡片阴影的深浅程度 |
| 按钮圆角 | 工具按钮的圆角半径（默认 4px） |
| 卡片圆角 | 分类卡片的圆角半径（默认 6px） |

**颜色选择器：** 点击任意颜色属性行，会弹出专业级颜色选择器，包含：
- **左侧调色板：** 98 色预设网格（8 色灰阶 + 18色相 × 5 明度），快速选色
- **右侧精细调节：** SV 色盘 + 色相滑条 + 明度滑条
- **顶部信息栏：** 颜色预览块 + HEX 输入框 + RGB 数值显示

### 2.6.3 主题导入与导出

自定义主题支持 **导入/导出** 功能，方便在多台电脑间共享主题配置：

- **导出：** 在编辑器中点击「导出主题」，将当前主题保存为 `.json` 文件
- **导入：** 点击「导入主题」，选择已有的 `.json` 主题文件加载

主题配置文件自动保存位置：安装目录下 `工具箱设置.json` 和 `自定义主题.json`，无需手动管理。
